## $100プランでできること

Claude Code Max 5xプラン（$100/月）は、日常的な経営運用に十分な利用量がある。

**1日の使い方の例:**
- 朝: 全部門の状態確認・朝ダイジェスト生成（5分）
- 午前: 開発タスク（コードレビュー、バグ修正）（2時間）
- 午後: マーケ（SNS投稿ドラフト生成）+ 営業（提案書作成）（1時間）
- 夕方: 経理（仕訳チェック）+ 法務（契約書ドラフト）（30分）

**$200プラン（Max 20x）が必要なケース:**
- サブエージェントを多用する並列処理

まずは$100で始めて、利用量の限界を感じたら$200に上げればいい。

## コスト最適化の3原則

### 1. サブエージェントを賢く使う

サブエージェントは便利だが、トークン消費が多い。

```markdown
## CLAUDE.mdでの設定例
- 単純なファイル読み書きはサブエージェントを使わず直接実行
- 3ファイル以上にまたがる調査はサブエージェントに委任
- レポート生成はSonnetモデルを指定してコスト削減
```

### 2. CLAUDE.mdでコンテキストを最小化

CLAUDE.mdが長すぎると、毎回のトークン消費が増える。

- CLAUDE.mdは200行以内に収める
- 詳細は別ファイル（`.claude/agents/`）に分離
- 参照ファイルのパスだけ渡し、中身は必要時にのみ読む

### 3. バッチ処理でまとめる

同じ種類のタスクはまとめて処理する。

```
❌ 記事1本ずつ生成 × 5回 → 5セッション分のコンテキスト
✅ 「記事5本をまとめて生成」 → 1セッション分のコンテキスト
```
