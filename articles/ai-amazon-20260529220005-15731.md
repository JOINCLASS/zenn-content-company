## はじめに：なぜ競合分析からやるのか

僕は今、AIで会社経営を完全自動化しながら、Zenn / Kindleで技術書・ビジネス書を継続出版しています。直近の月次でいうと、2026年3月の書籍売上はZenn ¥14,756 + Amazon ¥6,647で合計 ¥21,403。受託売上¥193万と合わせて月商¥195万ほど。「AI経営」を自分自身で実証している会社です。

そんな当社で次に出そうとしているのが **『スキルゼロから月3万円稼ぐAI副業スタートガイド』** という新刊。ところがこのジャンル、Amazonで検索すると分かりますが**完全な激戦区**です。匿名著者の薄い本から、KADOKAWAの10万部突破ベストセラーまで、玉石混交で並んでいる。

「とりあえず書いてみる」では確実に埋もれる。そこで出版部門エージェントに競合分析をやらせ、僕自身で勝ち筋を整理しました。本記事はその結果です。

## 1. 上位10冊を並べてみて分かったこと

Amazonの「AI 副業」上位を調べると、目立つのはこのあたり。

- **#1『2ヶ月で月30万円を実現する 超初心者でも稼げるAI活用法』**（あべむつき / KADOKAWA, ¥1,980, 304p, 10万部突破）
- **#2『毎月10万円をAIに稼いでもらう! ChatGPT副業の教科書』**（ChatGPTビジネス研究会 / 宝島社, ¥1,430, 159p）
- **#3『知識ゼロからの生成AIを活用した不労所得マシンの作り方』**（北村拓也 / 秀和システム, ¥1,760）
- **#4『ChatGPT副業で月3万円稼ぐ完全ロードマップ』**（ぴーちゃん, Kindle低価格）
- **#5『世界で一番やさしいインスタ×AI副業の教科書』**（じゅん, Kindle）

価格帯はKindleでだいたい¥980〜¥1,980。ページ数は150〜300pにバラついています。

ここで面白いのが、**売れている本ほど「具体的なロードマップ」と「著者の顔」が前面に出ている**ということ。KADOKAWAの#1は「LINEスタンプから始める」「穴埋め式プロンプト」「即日収益」という極めて具体的な導線で構成されています。一方、星1〜2が多い本は揃いも揃って匿名著者・薄いページ数・抽象論。

## 2. 低評価レビューに「読者の本音」が全部書いてある

低レビューを読み込むと、AI副業本のアンチパターンが綺麗に浮かびます。

### A. 「ネットで拾える情報の使い回し」
> 159ページで¥1,430は割高。ネットで無料で拾える情報ばかり

### B. 「いかにもAIで書いた感」
> 文章がのっぺりしていて読みにくい。具体的な数字や事例がなく、抽象的な話ばかり

### C. 「古い情報・再現性なし」
> 2023年時点の情報。2025年では競争が激しすぎて同じ方法は通用しない

### D. 「タイトル詐欺」
> 月30万円のタイトルだが、再現できるのは月3万円程度

### E. 「LINE誘導が主目的」
> QRコードがやたら多く、LINEコンサル誘導が主目的

つまり読者は、**「匿名・AI生成・薄い・古い・誘導」**の5点に強烈なアレルギーを持っている。逆に言うと、ここを全部潰した本を出せば、それだけで上位互換になり得る。

## 3. 競合に欠落しているもの（=市場ギャップ）

僕が分析していて「これは全員が書いていないな」と感じたのは次の8点でした。

| ギャップ | 読者が求めているのに供給されていないもの |
|---|---|
| リアルな収益証拠 | 著者自身の月次売上スクショ、P&L |
| 失敗事例・撤退判断 | 「どこで諦めるか」の基準 |
| ジャンル選定の現実直視 | 「ライティング代行はもう稼げない」と言える本がない |
| 2026年版ツール比較 | ChatGPT以外（Claude / Perplexity / Dify）の使い分け |
| 月3万→月10万のスケール | 達成後どう伸ばすかが空白 |
| 税務・確定申告 | 20万円ルールや経費計上 |
| クライアント獲得の実戦術 | ランサーズ/ココナラの受注文章例 |
| 著者の信頼性 | 実際にAI経営でいくら稼いでいるか |

ここで僕にとってラッキーだったのは、上記の多くが**当社の実体験そのもの**だということです。

## 4. 当社が持っている「他にない武器」

棚卸ししてみたら、当社にはこれだけのリアルデータがあります。

- 月商¥195万円（うち書籍¥21K + 受託¥193万）
- AI運用コストは月**約¥20,000のClaude Code Max**だけ
- 自動化率98%、launchdジョブ17本、SNS自動配信27件/日
- 10事業同時運用で**9事業が売上ゼロ**になった失敗
- AI出力を確認せず送信して**敬語が崩壊した営業メール**事件
- AIが`git push --force`を勝手に実行して**ブランチが消えた**事故
- CLAUDE.mdを1,000行超にしたらAIが**指示と逆の動作**を始めた件
- HPリニューアル営業に21社かけて**成果ゼロで撤退**した話
- Zennのレートリミットで複数回デプロイ失敗
- macOSでcronが動かずlaunchdに全面移行した話

副業ジャンルの本でこれらを書いている著者は、僕の知る限り**ゼロ**です。「うまくいった話」だけでなく「9事業ぶっ潰した」まで書ける本は強い。

CEOの判断ログにも残していますが、僕の経営方針はシンプルで「**AIは実行に使う。判断は人間。**」というもの。これは月200万を1人で回す上での核となる考え方で、副業本にも転用できます。

## 5. 勝ち筋の整理：差別化マトリクス

競合分析を踏まえ、新刊では次の5点で差別化します。

1. **著者の実績を冒頭に置く** — 月商¥195万の実数値・スクショで信頼を担保
2. **失敗事例と撤退ラインを章として独立させる** — 他本が触れない領域
3. **2026年最新ツール比較** — Claude / Dify / Perplexityまで含める
4. **税務・確定申告を1章使って解説** — 副業初心者が最も不安な部分
5. **価格は¥980〜¥1,280** — 競合と同等〜やや安く、内容で圧倒

ページ数は200〜250p、文字数6〜8万字、章数12。「薄すぎ」批判（150p以下）を絶対に出さない設計です。

## 6. この分析プロセス自体をAIで回している話

最後にメタな話を1つ。この競合分析、僕は**Amazon検索・レビュー読み込み・章構成提案までを出版部門エージェントに委任して、最終判断だけ自分でやっています**。所要時間は実質1時間弱。

朝の5分で全部門を把握して承認ボタンを押すだけ — これが当社の経営スタイルで、出版領域も例外ではありません。「仕組みの初期投資は大きい。だが一度作れば自動で回る。」この言葉どおり、本1冊出すごとに分析コストはほぼゼロに収束していきます。
