# Claude Code 利用同意書

私は、Claude Code（Anthropic社提供のAIコーディングツール）の利用にあたり、
以下の事項を理解し遵守することに同意します。

## 理解事項

1. Claude Codeに入力したプロンプトおよびClaude Codeが読み取った
   ファイルの内容は、Anthropic社のAPIサーバーに送信されます。

2. 当社は [Business/Enterprise/API] プランを利用しており、
   送信されたデータはAIモデルのトレーニングには使用されません。

3. 送信されたデータは安全性評価の目的で一定期間保持された後、
   削除されます。

## 遵守事項

1. 「Claude Code企業利用ポリシー（SEC-AI-001）」を遵守します
2. 個人情報や認証情報をClaude Codeに入力しません
3. IT部門が配布したPermissions設定を変更しません
4. Claude Codeの出力は必ずレビューしてから使用します
5. セキュリティインシデントが発生した場合は直ちに報告します
