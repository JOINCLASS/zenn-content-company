# 営業業務の自動化 -- 日程調整・フォロー・CRM連携

## はじめに -- 営業担当者は「売る作業」に何%の時間を使えているか

営業担当者の仕事は「売ること」だ。しかし実際には、営業担当者の1日の大半は「売る以外の作業」に費やされている。

営業支援ツール各社の調査によると、営業担当者が実際に顧客と対話している時間（商談、プレゼン、電話）は、**全労働時間のわずか28%**。残りの72%は、データ入力、メール対応、日程調整、報告書作成、社内会議などの間接業務だ。

:::message
**筆者の実践メモ**
筆者はAutonomosとSendGridを連携させ、1日27件のSNS投稿を自動配信している。ただし、営業の本質は「信頼構築」であり、自動化できるのは認知拡大の部分だけである。
:::

中小企業の営業担当者に限れば、この比率はさらに悪い。専任のアシスタントがいないため、見積書の作成からCRMへの入力まで、すべてを1人でこなす必要があるからだ。

本章では、営業の間接業務を3つの領域に分けて自動化する方法を解説する。

```
営業業務の自動化 3つの領域:
  1. 日程調整の自動化 -- メールの往復を0にする
  2. フォローアップの自動化 -- 追客の抜け漏れを防ぐ
  3. CRM入力の自動化 -- データ入力をゼロに近づける
```


## 領域1: 日程調整の自動化

### 現状の課題

「来週、打ち合わせのお時間をいただけますか？」

このメールから始まる日程調整は、平均して**3〜5往復のメールのやり取り**が必要だ。候補日を提示する → 相手が都合の悪い日を返す → 別の候補を出す → ようやく確定する。この1件の調整に15〜30分かかる。

週に5件の商談があれば、日程調整だけで月間5〜10時間を消費する。

### 解決策: 日程調整ツールの導入

日程調整ツールを使えば、**メールの往復がゼロ**になる。仕組みは以下の通りだ。

1. 自分のGoogleカレンダー（またはOutlookカレンダー）と連携する
2. 空き時間が自動でツールに反映される
3. 予約ページのURLを相手に送る
4. 相手がURLを開き、空いている時間帯から選んで予約する
5. 両者のカレンダーに自動で予定が登録される
6. 確認メールが自動送信される

### 主要ツールの比較

| ツール | 月額費用 | 特徴 | 日本語対応 |
|--------|---------|------|----------|
| Calendly | 無料〜$12 | 世界最大手。連携豊富 | 予約ページは日本語可 |
| TimeRex | 無料〜980円 | 日本製。日本の商慣習に合う | 完全日本語対応 |
| Spir | 無料〜800円/ユーザー | 日本製。グループ調整が得意 | 完全日本語対応 |
| Cal.com | 無料（オープンソース） | セルフホスト可能。カスタマイズ性高 | コミュニティ翻訳あり |
| 調整アポ | 無料〜 | 日本製。シンプルで使いやすい | 完全日本語対応 |

### 導入手順（TimeRexの場合）

**手順1: アカウント作成とカレンダー連携**
1. timerex.net にアクセスし、Googleアカウントでサインアップする
2. Googleカレンダーとの連携を許可する
3. 既存の予定が読み込まれ、空き時間が自動判定される

**手順2: 予約ページの設定**

1. 「新しい予約ページを作成」をクリック
2. ミーティングの種類を設定する
   - 名前: 例）「30分ミーティング」
   - 所要時間: 30分/60分/90分
   - 場所: オンライン（Zoom/Google Meet）/対面
3. 予約可能な時間帯を設定する
   - 曜日: 月〜金
   - 時間: 10:00〜12:00、14:00〜17:00（昼休みを除外）
   - バッファ: 前後15分（移動時間や準備時間）
4. Zoom/Google Meet連携を設定する（ミーティングURLが自動生成される）

**手順3: 予約ページのURLを活用する**

設定が完了すると、以下のような予約ページのURLが生成される。

```
https://timerex.net/s/yourname/meeting30
```

このURLをメールの署名やメールテンプレートに組み込む。

**メール文例:**
```
ご都合のよい日時をお選びください。
▶ 日程調整: https://timerex.net/s/yourname/meeting30

上記URLから空き時間をご確認いただき、
ご希望の時間帯をクリックしてご予約ください。
```

### 高度な活用: ラウンドロビンと自動割り振り

営業チームが複数名いる場合、「ラウンドロビン」機能を使えば、予約を自動的にチームメンバーに振り分けることができる。

例えば、Webサイトに「無料相談を予約する」ボタンを設置し、顧客が予約すると、空いている営業担当者に自動で割り振られる。手動で担当を決める手間がなくなり、予約から商談までのリードタイムも短縮される。

### 効果の目安

| 指標 | 導入前 | 導入後 |
|------|--------|--------|
| 1件の日程調整にかかる時間 | 15〜30分 | 0分（自動） |
| 日程調整のメール往復回数 | 3〜5往復 | 0往復 |
| ダブルブッキングの発生 | 月1〜2回 | 0回 |
| 月間の日程調整工数 | 5〜10時間 | ほぼ0 |


## 領域2: フォローアップの自動化

### 現状の課題

営業の成約率を最も大きく左右する要素は何か。それは「フォローアップのタイミングと頻度」だ。

業界の調査によると、リード（見込み客）への初回連絡が5分以内の場合、30分後の連絡と比較して**成約率が21倍**になる。しかし、多くの中小企業では、問い合わせへの返信に平均42時間かかっている。

また、見込み客の80%は「5回以上のフォローアップ」の後に購入を決定するが、営業担当者の44%は1回のフォローで諦めてしまう。

問題は、営業担当者のやる気がないことではない。**フォローすべき相手とタイミングを管理しきれない**のだ。

### 解決策: シーケンス（自動フォロー）の構築

「シーケンス」とは、あらかじめ設定した条件とタイミングで、自動的にメールを送信する仕組みだ。

### HubSpot CRM（無料）を使ったフォロー自動化

HubSpotは世界最大級のCRMで、**無料プランでも十分に実用的**だ。有料版ではシーケンス機能（完全自動のメール送信）が使えるが、無料版でも「タスクの自動リマインド」で同等の効果が得られる。

**手順1: HubSpot CRMの初期設定**

1. hubspot.com でアカウントを作成する
2. 会社情報を登録する
3. Gmailまたは Outlook を連携する
4. 取引パイプライン（商談のステージ）を設定する

**パイプラインの設定例:**
```
ステージ1: 問い合わせ受領
  ↓
ステージ2: ヒアリング済み
  ↓
ステージ3: 提案済み
  ↓
ステージ4: 見積提示済み
  ↓
ステージ5: 交渉中
  ↓
ステージ6: 成約 / 失注
```

**手順2: フォローアップのワークフローを設計する**

各ステージに対して、フォローアップのルールを決める。

| ステージ | フォロータイミング | アクション |
|---------|-----------------|----------|
| 問い合わせ受領 | 5分以内 | お礼メール + ヒアリング日程調整のURL送付 |
| ヒアリング済み | 翌営業日 | ヒアリング内容のサマリ + 次のステップを提案 |
| 提案済み | 3日後 | 「提案書はご確認いただけましたか？」メール |
| 見積提示済み | 5日後 | 「ご不明点はございませんか？」メール |
| 見積提示済み | 10日後 | 「期限が近づいています」リマインド |
| 失注 | 30日後 | 「その後いかがですか？」再アプローチ |

**手順3: メールテンプレートを作成する**

各フォローメールのテンプレートをHubSpotに登録する。

**テンプレート例: ヒアリング後のフォロー**
```
件名: 本日のお打ち合わせのまとめ

{{contact.firstname}} 様

本日はお忙しい中、お時間をいただきありがとうございました。

お打ち合わせの内容をまとめましたので、ご確認ください。

【ヒアリング内容】
- 課題: {{カスタムプロパティ.課題}}
- ご希望: {{カスタムプロパティ.希望}}
- ご予算: {{カスタムプロパティ.予算}}

【次のステップ】
上記の内容をもとに、{{日付}}までに提案書をお送りいたします。

ご不明点がございましたら、お気軽にご連絡ください。
```

**手順4: タスクの自動リマインドを設定する**

HubSpotの無料版でも、取引のステージが変更されたときに自動でタスク（リマインド）を作成できる。

1. 「自動化」→「ワークフロー」を開く（無料版では「シンプルワークフロー」が利用可能）
2. トリガー: 「取引のステージが"提案済み"に変更されたとき」
3. アクション: 「3日後にタスクを作成 — "提案のフォローアップ連絡"」

これにより、フォローの「抜け漏れ」がなくなる。

### 効果の目安

| 指標 | 自動化前 | 自動化後 |
|------|---------|---------|
| リードへの初回連絡速度 | 平均42時間 | 5分以内 |
| フォローアップの実施率 | 40%（2回目以降は減少） | 95%（自動リマインド） |
| 商談の成約率 | 15% | 22〜28% |
| フォロー管理の工数 | 月8〜12時間 | 月2〜3時間 |


## 領域3: CRM入力の自動化

### 現状の課題

CRM（顧客管理システム）は、正しく使えば営業活動を大きく強化するツールだ。しかし、多くの中小企業ではCRMが「データ入力の負担」として嫌われている。

よくある声:
- 「商談のたびにCRMに入力するのが面倒」
- 「入力する時間があったら、もう1件電話したい」
- 「結局Excelのほうが楽」

この問題の本質は、CRMの仕組みではなく、**データ入力のプロセス**にある。入力を自動化すれば、CRMは「負担」から「武器」に変わる。

### 自動化の手法

**手法1: メールからの自動取込**

HubSpotのGmail/Outlook連携を使えば、メールのやり取りが自動でCRMに記録される。

設定手順:
1. HubSpotにGmail/Outlookを連携する
2. 「ログ設定」で「送受信メールを自動でログする」をオンにする
3. CRMに登録されている連絡先とのメールが、自動で取引履歴に記録される

これだけで、「メールの内容をCRMに転記する」作業がゼロになる。

**手法2: 名刺スキャンからの自動登録**

展示会や交流会で受け取った名刺を、スマホで撮影するだけでCRMに登録する。

おすすめツール:
- **HubSpotモバイルアプリの名刺スキャン機能**（無料）
- **Sansan**（名刺管理の定番。月額費用はやや高い）
- **Eight**（無料の名刺管理アプリ。CSVエクスポートでCRMに連携可能）

**手法3: 商談メモの音声入力**

商談後の記録は、テキスト入力ではなく音声入力を使う。

1. 商談終了後、スマホの音声メモで商談内容を口頭で記録する（2〜3分）
2. 音声データをWhisper API（OpenAI）またはGoogle Cloud Speech-to-Textでテキスト化する
3. テキスト化されたメモをChatGPT/Claudeに整理させる（顧客名、課題、次のアクション等に分類）
4. 整理されたデータをCRMに登録する

ZapierやMakeを使えば、「音声ファイルをGoogle Driveに保存 → 自動でテキスト化 → AIで整理 → HubSpotに登録」という一連のフローを自動化できる。

**手法4: Webフォームからの自動登録**

Webサイトの問い合わせフォームや資料請求フォームの入力内容を、自動でCRMに登録する。

HubSpotには無料のフォームビルダーが付属しているため、このフォームを使えば入力内容が自動的にCRMのコンタクトとして登録される。

既存のフォーム（Googleフォーム、WordPress Contact Form等）を使っている場合は、Zapierで連携できる。

```
Googleフォームに回答が来る
  → Zapier が検知
  → HubSpot CRM に新しいコンタクトを作成
  → Slackの営業チャンネルに通知
```

### 効果の目安

| 指標 | 自動化前 | 自動化後 |
|------|---------|---------|
| CRMへのデータ入力工数 | 月7〜10時間/人 | 月1〜2時間/人 |
| CRMのデータ完全性 | 60%（入力漏れが多い） | 95%（自動取込） |
| 名刺の処理時間（100枚） | 5時間 | 30分 |
| 商談記録の入力時間 | 1件15分 | 1件3分（音声入力+AI整理） |


## 営業チーム全体の導入ステップ

### Week 1: 日程調整ツールの導入

1. TimeRex（または他のツール）のアカウントを作成する
2. チーム全員のカレンダーを連携する
3. 予約ページのURLを作成する
4. メール署名とテンプレートにURLを追加する

**この週の成果:** 日程調整のメール往復がゼロになる

### Week 2: CRMの導入と基本設定

1. HubSpot CRMのアカウントを作成する
2. 既存の顧客データをインポートする（ExcelのCSVで可能）
3. 取引パイプラインを設定する
4. メール連携を設定する

**この週の成果:** 顧客情報が一元管理され、メールが自動で記録される

### Week 3: フォローアップの自動化

1. フォローメールのテンプレートを作成する（5〜10パターン）
2. ステージ変更時のタスク自動作成を設定する
3. チーム全員にフォロールールを共有する

**この週の成果:** フォローの抜け漏れがなくなる

### Week 4: 運用の最適化

1. 1ヶ月間のデータを分析する（成約率、フォロー回数、リードタイム）
2. テンプレートの改善（開封率、返信率を確認）
3. パイプラインのステージを実態に合わせて調整する

**この週の成果:** データに基づいた営業改善が始まる

### 費用の総額

| 項目 | 月額費用 |
|------|---------|
| 日程調整ツール（TimeRex） | 無料〜980円 |
| CRM（HubSpot無料版） | 0円 |
| ワークフロー自動化（Zapier/Make） | 約3,000〜4,500円 |
| 生成AI API（商談メモの整理等） | 約2,000〜5,000円 |
| **合計** | **月額5,000〜11,000円** |

月額約1万円の投資で、営業チーム全体で月間30〜50時間の工数を削減。その時間を「顧客との対話」に回すことで、成約率の向上が期待できる。
